Social features
===============

*This feature is only available for customers with
`|acquia-product:bc| </dam/brand-connect>`__.*

Administrators can enable liking, commenting, and view and download
counts of assets in |acquia-product:bc|, which helps bring successful
and popular assets to the forefront.

This document covers the following topics:

-  `Turning on social features <#turn-on>`__
-  `Using social features <#use>`__

.. _turn-on:

Turning on social features
--------------------------

To turn on the social features of |acquia-product:dam|:

#. Log in to |acquia-product:dam| and then click **Brands** in the top
   navigation.
#. Click **Edit** for the **|acquia-product:bc| portal** you want to
   configure.
#. In the top navigation, select **Settings**.
#. Check the box next to **Enable/Disable Social** to enable the social
   features. When enabled, users may like and comment on assets.
#. Select the check box next to **Enable/Disable download and view
   count** to enable download and view counts. When enabled, users can
   see how many times an asset has been viewed and downloaded.
#. Click **SAVE** when finished.

.. _use:

Using social features
---------------------

Once enabled, users can use the various social features.

-  *Like*: Users may like assets by clicking the |Like icon| icon on the
   asset thumbnail or asset details page.
-  *Comment*: Users can comment by clicking the |Comment icon| icon on
   the asset thumbnail or the Comments section of the asset details
   page. All comments will be visible to users with view permissions to
   that asset.
-  *Views*: The view count will display next to the |Views icon| icon on
   the thumbnail and asset details page.
-  *Downloads*: The number of downloads will display next to the
   |Downloads icon| icon on the asset thumbnail and details page.
