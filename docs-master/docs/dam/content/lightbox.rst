Lightboxes in |acquia-product:dam|
==================================

You can use lightboxes to add assets from multiple folders to one
central location.

The following terms may help you as you use the information on this
product documentation page:

-  **Lightbox toolbar** - Displays at the bottom-left of the screen.
   Click the lightbox icon |Lightbox icon| on the lightbox toolbar to
   create a new lightbox or access an existing lightbox.
-  **Active lightbox** - The lightbox that you are currently adding and
   removing assets from. Only one lightbox can be active at a time — its
   name will display on the bottom-left of the page and, when reviewing
   the lightbox menu, will be highlighted in blue.
-  **Lightbox view** - Allows you to view, share, collaborate, print,
   duplicate, and download a lightbox. To access, click the lightbox
   toolbar, and then click the lightbox that you want to view.

.. admonition:: Notes

  -  Inactive assets will not be added to a lightbox.
  -  Collaborators will `receive a notification </dam/access/alerts>`__ if
     the lightbox is modified or a comment is added.
  -  |acquia-product:dam| displays lightboxes in alphabetical order in the
     lightbox menu.

.. _example:

Example use cases
-----------------

-  A user adds the imagery and data sheets for three different products
   to a lightbox so they can email a download link to an agency.
-  A graphic designer is choosing between a few images for the new
   product brochure and wants to receive feedback from the art director.
   Once the images are added to the lightbox, the graphic designer
   invites the art director to collaborate and comment on the
   selections.
-  A sales person creates a lightbox of frequently used customer logos
   for easy access.
-  A marketing team can collaborate on a batch of images where they can
   comment and decide which assets are to be used for a project.

.. _create:

Creating a lightbox
-------------------

To create a new lightbox for collecting your assets, complete the
following steps:

#. Sign in to |acquia-product:dam|, and then click the lightbox icon
   |Lightbox icon| at the bottom of the page.
#. Click the plus icon ( **+** ) on the top-right of the lightbox menu.
#. Enter the name, project (optional), and description (optional).
#. Select the **Switch** check box for the new lightbox to make it
   your active lightbox. |br|
   Only one lightbox can be active at a time – its name will display
   on the bottom-left of the screen and, when reviewing the lightbox
   menu, it will be highlighted in blue. |br|

   |Lightbox list|

#. Click **Save**.

.. _add:

Adding assets to a lightbox
---------------------------

Use one of the following methods to add assets to an active lightbox:

-  Point to the asset that you want to add, and then click the lightbox
   asset icon |Lightbox asset| on the asset thumbnail.
   Repeat this step for all assets that you want to add.
-  Select one or more assets that you want to add, click the lightbox
   asset icon |Lightbox asset| on the actions toolbar, and then click
   **Add**.

After an asset is added to your active lightbox, it will display a check
mark icon |Check mark icon| instead of the lightbox asset icon |Lightbox
asset|.

.. _remove:

Removing assets from a lightbox
-------------------------------

You can remove assets from a lightbox by using one of the following
methods:

-  Click the check mark icon |Check mark icon| on the thumbnail to
   remove assets from your active lightbox.
-  From the asset thumbnail view, select one or more assets to remove,
   click the lightbox asset icon |Lightbox asset| on the actions
   toolbar, and then click **Remove**.
-  From the lightbox view, select one or more assets to remove click the
   lightbox asset icon |Remove from lightbox|, click Remove, and then
   click **Remove** to confirm.

.. _download:

Downloading a lightbox
----------------------

Use one of the following methods to download assets from a lightbox:

-  Click the lightbox toolbar icon |Lightbox icon| on the bottom of the
   page. Point to the lightbox that you want to download in the lightbox
   menu, and then click the download icon |Download|.
   To customize the size and file type of the downloaded asset, click
   the **Desired preset** dropdown menu.
-  In the lightbox view, select one or more assets that you want to
   download, and then click **Download**.
   To customize the size and file type of the downloaded asset, click
   the **Desired preset** dropdown menu.
-  In the lightbox view, use the following steps to download assets to
   an FTP:

   #. Click the share |Share icon| icon on the actions toolbar, and then
      click **FTP lightbox download**.
   #. Enter the server address, username, password, and remote directory
      of the FTP. You can also select from previously used servers.
   #. If you want to customize the size and file type of the downloaded
      asset, click the **Download size** list.
   #. Click **Send**.

.. _collaborate:

Collaborating on a lightbox
---------------------------

You can invite other |acquia-product:dam| users to collaborate on a
lightbox. Collaborators can add comments and modify the lightbox.

To invite users to collaborate, complete the following steps:

#. In the lightbox view, click the add collaborator icon |Add
   Collaborator| in the left column:
   |Collaboration menu|
#. Enter one or more email addresses and a message (optional).
#. If you want the recipients to be able to modify the lightbox, select
   the **These users can modify** check box. This allows these users to
   edit lightbox information and add and remove assets.
#. Click **Share**.

To edit collaborators:

#. In the lightbox view, click the add collaborator icon |Add
   Collaborator| in the left column:
   |Collaboration menu|
   A lightbox collaborator icon |Lightbox Collaborator| indicates that a
   user with that email address exists in |acquia-product:dam|, while a
   **(?)** icon indicates that the email address is not connected to a
   registered |acquia-product:dam| user.
#. Edit the collaborators:

   #. Select the check box to allow/restrict the collaborators’
      permission to modify the lightbox.
   #. Click **X** to remove collaborators.

.. _comments:

Commenting on lightboxes
------------------------

You can add comments to the lightbox or individual assets.

-  *Lightbox comment* - Add the comment in the collaborate section in
   the left column of the lightbox view:
   |Comment here|
-  *Asset comment* - In the lightbox view, click the asset once to
   select it, and then add the comment in the asset details panel on the
   right:
   |Asset comment|

Assets with comments will display a comment bubble icon |Comment bubble|
in the top left corner of the asset displaying the number of comments.
Comments will be visible only to collaborators added to the lightbox:

|Asset with comment|

By default, users will receive an in-system notification and email when
a collaborator comments or modifies a lightbox. For more information
about notifications, see (`Obtaining notifications and
alerts </dam/access/alerts>`__.

.. _options:

Additional lightbox options
---------------------------

-  *Switch lightbox to active lightbox* - Click the |Lightbox icon|
   **Lightbox** icon on the bottom-left of the page. Hover your mouse
   over the lightbox you want to activate, then click the |Active
   Lightbox| **Active Lightbox** icon.
-  *Delete* - In the lightbox view, click the delete lightbox icon
   |Delete Lightbox| on the actions toolbar.
-  *Edit lightbox information* - Click the lightbox icon |Lightbox icon|
   on the bottom-left of the page. Point to the lightbox that you want
   to edit, and then click the edit lightbox icon |Edit lightbox| to
   update the name, project, and description.
   These fields are also editable in the left column of the lightbox
   view.
-  *Duplicate a lightbox* - In the lightbox view, click the duplicate
   lightbox icon |Duplicate lightbox| in the actions toolbar. The
   duplicate lightbox will appear in the lightbox menu, with *Copy of*
   preceding the name.
   For example, a duplication of the *Spring Collection* lightbox would
   be called *Copy of Spring Collection*.
-  *Print a lightbox* - In the lightbox view, click the print icon
   |Print a lightbox| in the actions toolbar. You can select a simple
   view that includes the assets’ thumbnail and filename, or a detailed
   view that also includes the assets’ metadata.
-  *Share a lightbox* - In the lightbox view, click the share icon
   |Share icon| in the actions toolbar. You can then click one of the
   following options:

   -  `Email link </dam/content/asset/share>`__
   -  `Email download </dam/content/asset/share>`__
   -  `Box </dam/integrate/box>`__
   -  `Dropbox </dam/integrate/dropbox>`__
   -  `FTP lightbox download </dam/content/lightbox>`__

-  *View all lightboxes* - Click the lightbox icon |Lightbox icon| on
   the bottom-left of the page, and then click **All lightboxes** in the
   lightbox menu. From this view you can `watch </dam/access/alerts>`__,
   duplicate, switch to, share, edit, or delete a lightbox.
