Developer role
==============

The developer role on |acquia-product:edg| should be assigned to users who are 
responsible for updating the development and staging environments with new and 
updated code for website development and testing.

.. note::  

   Users with the developer role must also have the |platform admin|_ role to 
   use the **Administration** admin menu link to access its available actions.

.. |platform admin| replace:: *Platform admin*
.. _platform admin: /site-factory/users/admin/platform-admin

To request that a user be given the platform admin role, contact Acquia
Support.

Users who have the developer role can perform the following actions on
|acquia-product:edg|:

-  Sign in to |acquia-product:edg|.
-  View the site update status for an environment.
-  View the environment's staging deployment status.
-  Deploy code updates to the development and staging environments for
   testing.
