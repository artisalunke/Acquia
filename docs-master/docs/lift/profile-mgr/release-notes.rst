Lift Profile Manager 1.67
--------------------------

*1 March 2018*

The |acquia-product:lpm| component of |acquia-product:cha| contains the
following updates:

Feature
~~~~~~~

-  **Learn more about your users and groups using the API** |br|
   The |acquia-product:liftapi| now includes the methods for
   `Users <http://docs.lift.acquia.com/profilemanager/#users>`__ and
   `Usergroups <http://docs.lift.acquia.com/profilemanager/#usergroups>`__,
   allowing you to use the API to learn more about your users in your
   custom applications.

Fixed issue
~~~~~~~~~~~~~~

-  On the Analytics page, when a user attempted to select a category in
   the report menu, |acquia-product:lpm| displayed an error message.

Lift Profile Manager 1.65
--------------------------

*7 February 2018*

The |acquia-product:lpm| component of |acquia-product:cha| contains the
following update:

Feature
~~~~~~~

-  **Managing custom data reporting is now available for general use** |br|
   Saving, modifying, and deleting custom data reports (introduced to
   some subscribers in version 1.64) are now available to all
   |acquia-product:lpm| subscribers that have the appropriate account
   access permissions. 
   `Learn more. </lift/profile-mgr/analytics/dashboards/custom>`__


Lift Profile Manager 1.64
--------------------------

*17 January 2018*

The |acquia-product:lpm| component of |acquia-product:cha| contains the
following update:

Feature
~~~~~~~

-  **Save, modify, or delete your custom data reports** |br|
   You can now save your custom reports for later reuse, without having
   to recreate them each time they are needed. |acquia-product:lpm| also
   allows you to modify your previously created reports, and to delete
   unneeded reports. 
   `Learn more. </lift/profile-mgr/analytics/dashboards/custom>`__
