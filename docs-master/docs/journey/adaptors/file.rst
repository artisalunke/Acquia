File-related adaptors
=====================

.. toctree::
   :hidden:
   :glob:

   /journey/adaptors/file/*

You can combine a third-party cloud service provider's file storage
system with |acquia-product:aj|, enabling you to store and retrieve
files from within your `graphs </journey/getting-started/graph>`__ based
upon logic specific to a customer journey. The following adaptors are
available for your use:

-  `Read </journey/adaptors/file/read>`__ - Download a file from
   cloud storage
-  `Write </journey/adaptors/file/write>`__ - Upload a file to cloud
   storage
