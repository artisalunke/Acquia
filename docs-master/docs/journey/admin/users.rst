Adding users to |acquia-product:aj|
===================================

.. toctree::
   :hidden:
   :glob:

   /journey/admin/users/*

.. container:: message-status

  This feature requires that you have the `Owner
  role </journey/admin/users/roles>`__ for your organization.

After signing in, go to the **Admin** page, and then click **Users** in
the left menu.

.. note::
  If the **Users** tab is not visible, but you believe you should have
  access to it, `contact Acquia Support </support#contact>`__.

Adding a user
-------------

To add a user, complete the following actions:

#. From the user details view, click the **Plus** icon ( |Plus icon| ).
   |acquia-product:aj| will display an **Add User** dialog box.
#. Enter the **Name** and **Email** address of the user to be added in
   the appropriate fields.

   .. important::
      The users are not restricted by domain, so please be careful when
      entering the email address.

#. Click **Create User**.

|New user information|

The user administration panel will display the new user's details. You
can now add a user to groups or projects, and set the user's role in the
project.

.. note::
  To add a user a to a group, use the Groups tab in the Admin screen. See
  `Creating and managing groups </journey/admin/groups>`__ for more
  information.

The user will receive an welcome email that includes the information
needed to sign in to |acquia-product:aj|.
