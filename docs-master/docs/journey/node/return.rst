Return node
===========

|Return node|

The Return node returns a value from a
`graph </journey/getting-started/graph>`__. Every graph must have at
least one Return node. A graph without at least one Return node is
invalid and will not return data.

The Return node can return a value from the schema, a public variable,
or a literal. The return value can be a either a primitive value or a
JSON object.
