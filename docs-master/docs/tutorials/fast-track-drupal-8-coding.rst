Tutorial Overview
-----------------

The purpose of the lessons in this tutorial is to explain Drupal 8
coding concepts to Drupal 7 developers.

These lessons contain a set of small learning cards which map
frequently-used Drupal 7 concepts or code snippets to their Drupal 8
equivalents.
