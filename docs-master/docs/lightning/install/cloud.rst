Installing |acquia-product:ld| on |acquia-product:ac|
=====================================================

.. raw:: html

   <div class="logo-pp">

.. raw:: html

   </div>

Hosting your |acquia-product:ld| website on |cloudlink|_ allows you to easily manage your
website's workflow across multiple environments using a scalable hosting
platform that's tuned for Drupal.

.. |cloudlink| replace:: \ |acquia-product:ac|\ 
.. _cloudlink: /acquia-cloud

.. note::  |acquia-product:ld| requires that you use Composer to properly manage 
   the website's codebase. For information about installing Composer and
   using it with your |acquia-product:ac| websites, see 
   `Using Composer with Drupal 8 sites </resource/composer>`__.

Procedure
---------

Use the `Creating a new application </acquia-cloud/create>`__
|acquia-product:ac| documentation page and follow its procedures to add
Drupal websites for your use. During the installation procedure, be sure
to select **Lightning** from the list of available
distributions.

For information about features available during the installation
process, see 
`Managing the additional Lightning features </lightning/install/features>`__.
