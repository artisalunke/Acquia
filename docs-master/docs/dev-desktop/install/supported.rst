Supported operating systems for |acquia-product:add|
====================================================

|acquia-product:add| has been tested with the following operating
systems. Earlier versions of these operating systems are not supported.

-  *macOS*

   -  OX X 10.12 Sierra
   -  OS X 10.11 El Capitan
   -  OS X 10.10 Yosemite
   -  OS X 10.9 Mavericks
   -  OS X 10.8 Mountain Lion
   -  OS X 10.7 Lion

-  *Windows (32- and 64-bit)*

   -  Windows 10
   -  Windows 8.0 and 8.1
   -  Windows Vista
   -  Windows 7
