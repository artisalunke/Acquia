Using the |acquia-product:ui|
=============================

The |acquia-product:ui| provides a set of tools that you can use to
manage your |acquia-product:ac| applications. These tools are available
by signing in at:

**https://cloud.acquia.com**

The toolbar at the top of the webpage provides the following options:

|Cloud Applications page|

-  **Develop** - The Applications page, where you can manage your
   |acquia-product:ac| applications and their infrastructure (code,
   environments, servers, databases, and more).
-  **Manage** - The Organizations page, where you can manage the teams,
   team members, roles, and permissions that govern access to your
   |acquia-product:ac| applications. You can also `view information
   about your Acquia subscriptions </acquia-cloud/subs>`__.
-  **Help Center** - A link to the Acquia Help Center, which gives you
   access to Acquia product documentation and to Acquia Support and TAM
   tickets.
-  **[your profile picture]** - Manage your Acquia profile.
