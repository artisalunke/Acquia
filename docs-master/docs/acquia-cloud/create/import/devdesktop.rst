Importing directly from |acquia-product:add|
============================================

|acquia-product:add| is a free xAMP stack installer or *DAMP stack*
installer, providing a full Drupal-specific stack that includes Apache,
MySQL, and PHP. It is a great tool to run and develop Drupal
applications locally on your computer. Since it is closely integrated
with |acquia-product:ac|, using |acquia-product:add| can be the easiest
way to import your application into |acquia-product:ac|. `Learn more
about |acquia-product:add| </dev-desktop>`__, or `download it
here <https://dev.acquia.com/downloads>`__.

To import your existing application to |acquia-product:ac| directly from
your installation of |acquia-product:add|, complete the following steps:

#. `Enable SSH access </acquia-cloud/ssh/enable>`__ to
   |acquia-product:ac|.
#. `Directly export your Drupal application into
   |acquia-product:ac| </dev-desktop/cloud>`__ using the **Host this
   site on Acquia Cloud** control in the |acquia-product:add| user
   interface.
