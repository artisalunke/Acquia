Acquia Journey adaptorsÂ¶

Acquia Journey provides a wide variety of adaptors, enabling you to interact with external systems from within your graphs. This allows you to drive your customer next best action based on specific information about a user.

Adaptors are created in the project editor, and each adaptor uses a pre-defined connection to do its work. Based on your connection type, the following adaptor types and methods can be created:

Connection Image Action / Method      
Acquia Lift Capture Decide Visitor Content
Database Read Write Listen Delete
Email   Write    
File Read Write    
Graph API     Listen  
Message Queue Read Write Listen  
REST Web Service GET POST PUT DELETE
SOAP Web Service Web service-dependent      
Twitter Read Write Listen  

Adding an adaptorÂ¶

To add an adaptor to your project, complete the following actions:

Note

Most adaptors require that you first create the connection that is required for the adaptor type that you want to create.

  1. In the Projects page, click the Project Editor icon on your projectâs card. Acquia Journey will display the Project Editor for your project.
  2. Click the action menu ( ) in the upper left of the page, and then click Create new.
  3. Enter a Name for the adaptor that describes its purpose.
  4. In the Adaptors section, click the adaptor type that you want to use.
  5. At the bottom of the page, click Create New Item. Acquia Journey will display the configuration page for your new adaptor.
  6. In the Adaptor Connection list, click the connection that you want to use for this adaptor.
  7. Continue to configure the adaptor based on its specific configuration instructions.
