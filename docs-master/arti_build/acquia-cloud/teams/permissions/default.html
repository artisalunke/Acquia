Default Acquia Cloud user permissions¶

Acquia Cloud user roles have multiple permission levels. These permissions are dependent on subscription type, and can be modified. You can check your role permissions against the default using this list:

  • Team Lead - Full access to subscriptions, can invite other members to the team
  • Senior Developer - Access to Dev, Stage, and Prod environments
  • Developer - Access to Dev and Stage environments
