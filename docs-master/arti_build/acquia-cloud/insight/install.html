Before you can access Acquia Insight, you have to connect your website to Acquia by installing and activating the Acquia Connector module. The |aic| is a Drupal module package that allows your websites to exchange information with Acquia.

Note

Enabling the Acquia Connector module in a local, development website can cause the website to be listed on your Acquia Cloud subscription page along with your other applications.

To install the Acquia Connector, complete the following steps:

  1. Sign in to your Drupal website as an administrator.

  2. Install and enable the Acquia Connector for your version of Drupal. (Drupal 8 instructions | Drupal 7 instructions)
    When enabling your Acquia Connector module in your Drupal website, enable the following modules on the Extend/Modules webpage, by Drupal version:
    • Drupal 8 â Acquia connector
    • Drupal 7 â Acquia Agent and Acquia Site Profile Information

    Note

    If youâre using the Lightning Drupal distribution, the Acquia Connector is already included, and you do not need to download it.

  3. Go to the Acquia Networ Connector configuration webpage using the appropriate method, based on your installed Drupal version:

    • Drupal 8 â In the admin menu, go to Configuration > Acquia Connector settings.
    • Drupal 7 â In the admin menu, go to Configuration > Acquia Subscription settings.

  4. If the Configuration page does not indicate that you are connected to an Acquia subscription, click Get connected, and then enter your Acquia email address and password.

    Note

    For multisite applications, it is helpful to enter a unique name in the Site Identification field. Acquia Cloud-hosted websites are provided with a name and machine name by default, which are displayed in Acquia Cloud interface pages, such as Stack Metrics, to help identify individual installations.

  5. Choose the subscription that you want to connect, and then click Submit. If you are using Acquia Connector 7.x-3.x, and your website is not hosted on Acquia Cloud, you must enter a name for your website under Site identification, and then click Save settings.

Adding Acquia SearchÂ¶

The Acquia Connector also includes the Acquia Search integration, a fully managed SaaS search offering that integrates seamlessly with Drupal applications. Built on the enterprise-grade Apache Solr search engine, Acquia Search is hosted on Acquia Cloud, providing a fully redundant infrastructure that ensures search is a highly available feature of your website. Learn more about Acquia Search.

Your website is now connected to Acquia. You may need to run cron on your website before it appears in Acquia Insight. For more information, see When the Acquia connector connects to Acquia. See also Using Insight to optimize your application.

Updating the Acquia ConnectorÂ¶

To get the greatest benefit from Acquia Cloud, you should keep the Acquia Connector module up to date with the latest stable version.

Important

When you upgrade the Acquia Connector module, be sure that you install the upgraded version in the correct location in your docroot directory. If you are using an Acquia-sponsored Drupal distribution, such as Lightning, the Acquia Connector module may be already installed in [docroot]/profiles/acquia/modules and not in [docroot]/sites/all/modules. Because of this you should upgrade the module in the location where it is installed. Having different versions of the module installed in more than one location can result in confusion or inconsistent behavior.
