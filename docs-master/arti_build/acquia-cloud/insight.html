Acquia Insight¶

Using Acquia Insight

  • Intro
  • Using
  • Alerts
  • Modules
  • Connector
  • Configure

Acquia Insight is a set of tools that evaluates your website’s performance, security, and search engine optimization to help you monitor and optimize your websites. Insight uses the |acquia-product:anc| module on your website to connect to the Acquia Insight service on a regular schedule, sending up-to-date information about the website’s code and configuration.

About the Acquia Connector¶

Before you can use Acquia Insight, you must install and enable the latest stable release of Acquia Connector. The |acquia-product:anc| is a Drupal module package that enables your sites based on Drupal 8 or 7 to exchange information with the Acquia Insight service. It consists of the following modules:

  • |acquia-product:anc| - Enables secure communication between your Drupal website and Acquia to monitor up-time, check for updates, and collect site information. For details about what information the Acquia Connector sends, see Sending information to Acquia.
  • |acquia-product:as| - (Drupal 7) Gets you started with |acquia-product:as|, Acquia’s plug-and-play search service.

In the 7.x versions, the Acquia Insight module is divided into separate Acquia Agent and Acquia Site Profile Information modules.

Acquia’s |acquia-product:ld| distrubution begins with the Acquia Connector modules already installed. This enables you to connect your site to Acquia Insight without installing any additional modules.

For more information about installing the Acquia Connector and how it works, see:
