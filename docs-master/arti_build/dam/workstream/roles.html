Workstream user roles and capabilities¶

This article covers the actions different Workstream users can complete. User roles are assigned in Acquia DAM, use the following to learn more about Configuring Users and Groups.

Only Admins can:¶

  • Configure Workstream Preferences
  • Create and edit Request Forms
  • Create and edit Approval Paths
  • Create Projects
  • Change Project permissions

Admins and Contributors can:¶

  • Submit a Request Form
  • Edit Tasks
  • Assign Tasks or Be a Task Assignee
  • Edit Projects
  • Be a Project Sponsor
  • View Tasks and Projects
  • Watch Tasks and Projects
  • Initiate Approvals

Admins, Contributors, End Users or Guests can:¶

  • Be an Approver
  • Proof (if enabled in the Approval settings)
  • Submit a request in |acquia-product:bc|
