Embeddable: Carousel¶

This feature is available only to Acquia DAM Professional subscribers.

Embeddables allow admins and contributors to create code that embeds assets on third-party platforms, including Wordpress, Drupal, and Brand Connect — essentially anywhere you can embed code.

The carousel embeddable allows you to schedule when and how long an asset displays, and what asset comes next.

For example, you want to display a holiday-themed promotional asset on your Wordpress-powered website during the holidays, and when the holidays are over you want to replace the asset with a different campaign asset for an upcoming product launch. To do this, you can create a carousel that displays the holiday asset from December 15 to January 1, which then gets replaced by the campaign asset.

Using the carousel, you can schedule as many assets as you want, as far into the future as you need.

There are two kinds of carousels available for your use:

  • Basic carousel - Embed an asset or group of assets that rotate and repeat during specific intervals of time (hours, days, weeks, months).
  • Advanced carousel - Instead of a group of assets that rotate for the same period of time, schedule individual assets to rotate in and out at specific times.

Creating a basic carousel¶

A basic carousel allows you to embed an asset or group of assets that rotate and repeat during specific intervals of time (hours, days, weeks, months). To create a basic carousel, complete the following steps:

  1. Sign in to Acquia DAM.

  2. Select one or more assets that you want to embed, click the pencil icon in the actions toolbar, and then click Embed assets.

  3. In the Carousel dialog box, configure your carousel with the following fields:

    • Carousel name - Enter a descriptive name for the carousel.
    • Add a description - Optionally, enter additional information to identify the carousel.
    • Carousel behavior - Click Basic.
    • Start - Skip this field to have the carousel start immediately after being created, or click Change to customize the start time.

  4. Click the Interval dropdown menu to select Hours, Days, Weeks or Months and enter the interval duration.

  5. Click the box next to Repeat if you want these assets to repeat their rotation.

  6. Click Continue.

  7. Click ADD ASSETS to add additional assets to your carousel.

  8. The top asset is your default asset, meaning it will display any time there is a gap in scheduling. To change the default, point to the current default image, and then click the pencil icon . Select a new default, and then click Insert.

  9. If you want to change an individual asset’s interval time, click the duration under the asset file name, update the interval and click Set.

  10. Click the Continue button.

  11. Select the preferred resolution Size, and then copy the Embed code.

You can now paste the embed code wherever you want to post the carousel.

Creating an advanced carousel¶

An advanced carousel allows you to schedule start and stop times and dates for individual assets, with no looped repeating.

  1. Sign in to the core application.
  2. Select the assets you want to embed (hold down Command key on a Mac or the CTRL key on a PC to select multiple assets) and click the Pencil icon on the actions toolbar. Select Embed assets.
  3. SETUP:
    1. Enter a carousel name and an optional description, then select Advanced.
    2. Click Change to customize the start time, or move on if you want it to start immediately.
    3. Click the Continue button.
  4. ASSETS:
    1. Click ADD ASSETS to add additional assets to your carousel.
    2. The top asset is your default asset, meaning it will display any time there is a gap in scheduling. To change the default, hover over the current default image and click the Pencil icon. Choose a new default and click Insert.
    3. Click Schedule next to each asset to set start and stop display dates and times. By default, the interval date will be one week.
    4. Click the Continue button.
  5. EMBED:
    1. Select the preferred resolution size, then copy the embed code.
    2. Paste the embed code wherever you want to post the asset.

Managing a carousel¶

Contributors may manage their own carousels, while admins may manage carousels created by all users.

  1. Sign in to the core application.
  2. At the bottom of the the left menu, click Embeddables.
  3. Hover over the embeddable you want to edit and click the Pencil icon. Select Edit to make changes or check settings.
  4. Edit the settings. Click the Close button when you’re finished.
