Integration: Box¶

The Box integration makes it easy to move files back and forth between Acquia DAM and Box. You can use it to quickly centralize assets stored in various personal or professional Box accounts, and to distribute files to people who do not have access to Acquia DAM.

This integration is available to Admin, Contributor, and regular users with download or upload permissions. You can transfer assets smaller than 5GB — assets greater than 5GB must use FTP.

Enabling the integration¶

The information in this section applies only to users that have `*Admin <#role>`__ role type permissions.*

  1. Sign in to Acquia DAM.
  2. Click the Settings icon in the top navigation, and then click System Preferences.
  3. In the Cloud Sharing Settings section, click the Cloud sharing channels field, and then click Box.
  4. Click Save.

Sending assets to Box¶

Users with download access can start sending assets to Box once the administrator has enabled the integration:

  1. Sign in to Acquia DAM.

  2. Click an asset to get to the asset details page.

  3. Click Show Sharing Options in the actions toolbar and select Box from the dropdown menu.

  4. The first time you send an asset to Box, you will need to click Add Integration.

  5. Sign in with your Box credentials.

  6. In the dialog box, create a new Box folder or choose an existing one.

  7. Click Send. The will file go to the selected folder.

You can use a a lightbox to send multiple assets to Box at once:

  1. Add the assets that you want to send to a lightbox.
  2. Click the Share icon in the actions toolbar, and then click Box.
  3. In the dialog box, create a new Box folder or choose an existing one.
  4. Click Send. The will file go to the selected folder.

Sending Box assets to Acquia DAM¶

To send assets or folders from Box to Acquia DAM, you will need to have upload permission in Acquia DAM.

  1. Sign in to your Box account.
  2. Go to https://app.box.com/apps and search for Webdam.
  3. Select the Webdam app and add it to your account.
  4. From the main page of your Box account, click the more actions icon next to the folder or asset that you want to send, and then click More Actions > Send file to Webdam.
  5. You will initially need to enter your Acquia DAM credentials.
  6. Choose the folder to send to. The asset will be queued for import.

The newly uploaded files will appear in the folder as Asset Processing while the preview thumbnail is generating.
