Creating metadata templates¶

Metadata Templates can be used to apply a set of metadata values to a group of assets upon upload or as a batch edit. This set of metadata can include specific values for active fields as well as keywords.

Create a metadata template¶

  1. Sign in to Acquia DAM, click the Settings icon, and select Metadata Schema.
  2. Click Metadata Templates in the left-hand navigation.
  3. Click the (+) Plus icon in the actions toolbar.
  4. Enter a template name and populate the values for any or all active metadata fields. (Read more about configuring the metadata schema.)
  5. Click Save.

Apply a metadata template to a folder¶

When a metadata template is applied to a folder, any assets uploaded into that folder will automatically adopt the metadata from the template. However, the metadata template will not be applied to the assets already residing in or moved to the folder.

There are two ways to apply metadata templates to a folder:

Option 1:

  1. Log into Acquia DAM, click the Settings icon, and select Metadata Schema.
  2. Click Metadata Templates in the left-hand navigation.
  3. Click the number under the Number of folders applied column. You can also click the Wrench icon, and choose Select folders.
  4. Click the box next to the folder(s) you want the metadata template applied to.
  5. To apply the metadata template to nested folders, switch the Apply to Nested toggle for the parent folder to Yes. (Note: The toggle will always default to No. The toggle can be updated when changing the metadata templates for a specific folder.)
  6. Click Save.

Option 2:

  1. Log into Acquia DAM.
  2. Click the downward-pointing triangle next to the appropriate folder in the folder tree and select Edit.
  3. At the bottom of the dialogue window, choose a template from the Apply metadata template dropdown. You can also choose to Apply settings to all nested folders with the checkbox.
  4. Click Save.

Apply a metadata template to an asset¶

Admins and contributors can apply a metadata template to assets.

  1. Log into Acquia DAM.
  2. Select the asset you want to edit.
  3. In the actions toolbar, click the Edit or Batch Edit icons, and select Edit metadata.
  4. Select a metadata template in the dropdown menu.
  5. You can then make changes to the values as needed.
  6. If the metadata should apply (will be added to any pre-existing metadata) to only one asset in the set, use the dropdown to Apply to this Asset Only.
  7. If the metadata should be apply to all assets in the set, use the dropdown to Append to all.
  8. If the metadata is replacing all existing metadata (overwrites any pre-existing metadata), select Replace All:
  9. Click Save.
