Using the Adobe Creative Cloud connector¶

This is an add-on feature. Contact your Account Manager to add this feature to your subscription.

Connector for Adobe Creative Cloud

  • Intro
  • 2018
  • 2017
  • 2015
  • Use
  • Uninstall

No more broken design links! When you use the Acquia DAM Connector for Adobe Creative Cloud, all of the files linked to your design are available to anyone with Acquia DAM. (Access depends on having permission to the folder(s) containing the assets.) This also reduces file size because there is no need to embed images.

For an overview of the Connector, see Connector for Adobe Creative Cloud.

Note

Acquia DAM Connector for Adobe Creative Cloud is an add-on feature. If you want to add this to your Acquia DAM subscription, contact Acquia Support for additional information.

Signing in to the Connector¶

  1. Open InDesign, Photoshop, or Illustrator.

  2. Create a new file or open an existing one.

  3. In the menu bar, click Window.

  4. From the Extensions section, click |acquia-product:dam| CC Connector.

  5. If you are not prompted to sign in, click the options icon in the upper-right of the Connector panel, and then click Authorize.

  6. Enter your Acquia DAM credentials, and then click Login.

  7. Click Yes to grant access to your account.

Using the Connector¶

  • Browse - Double-click the folders in the Connector. View the folder tree by clicking the folder name on the Connector header. To go back a level, click Home or a parent folder. You can also click the previous icon to go back to the previous folder.

  • Search - Click the magnifying glass icon, enter a search term, and then click Search. Set the toggle to green to refine your search to supported files.

  • Preview - Double-click an asset in the Connector to display a larger preview image.

  • Add - From the thumbnail view, drag and drop assets into the document from the Connector.

  • Upload document to |acquia-product:dam|

    1. Save the document to your computer.
    2. In the Connector, navigate to the folder where you would like to upload the document.
    3. Click the options icon on the top-right of the Connector panel, and then click Upload Document.

    Your file and any locally linked assets will now be saved in Acquia DAM for you or other team members to access. There is no need to package the file, because all of your links will pull linked files directly from Acquia DAM.

Additional information on using the Acquia DAM Connector in InDesign¶

The Acquia DAM Connector lets you drag and drop images from Acquia DAM into InDesign to create direct, URL-based links.

In addition to the previously-described functionality, you may also do the following:

Update resolution - Double-click the assets icon in the links panel to toggle between-low res and high-res assets. You can update all links at once by pressing Option (MacOS) or Alt (Windows) as you double-click the assets icon .

Updating version - If a new version of an asset has been added in Acquia DAM, the warning icon will be displayed in the Links panel. Double-click the icon to update to the latest version.

Moving assets in |acquia-product:dam| - There will be no impact on the links if an asset used in the InDesign document is moved in Acquia DAM.

If you need to distribute an InDesign to someone who does not have Acquia DAM, you can use the InDesign Package command with the option to include copies of the assets from Acquia DAM in the output file.
