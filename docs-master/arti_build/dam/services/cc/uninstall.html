Uninstalling the Adobe Creative Cloud connector¶

This is an add-on feature. Contact your Account Manager to add this feature to your subscription.

Connector for Adobe Creative Cloud

  • Intro
  • 2018
  • 2017
  • 2015
  • Use
  • Uninstall

Only one version of the Acquia DAM Connector can be active at a time. Because of this, you must remove any existing Acquia DAM Connector plug-ins from your InDesign Plug-Ins folder before installing a ZXP package that contains Acquia DAM Connector plug-ins.

After the uninstall is complete, you can install the most-recent version of Acquia DAM Connector for Adobe Creative Cloud.

Note

Because Adobe no longer supports CS6, users with the Acquia DAM Connector with CS6 may experience limited connectivity and support options.

macOS users¶

  1. Quit InDesign, Photoshop and/or Illustrator if it’s open.
  2. Open a Finder window.
  3. Open your Library folder.
    1. Click Go on the top navigation and choose Go to Folder.
    2. Enter ~/Library
  4. Navigate through the folder path to /Library/ApplicationSupport/Adobe/CEP/extensions and move the |acquia-product:dam|CCConnector file to the trash.
  5. Open your Applications folder.
  6. Navigate through the folder path to /Applications/Adobe InDesign CC 201#/Plug-Ins and move the SiliconConnector|acquia-product:dam| file to the trash.

Windows users¶

  1. Quit InDesign if it’s open.
  2. Navigate to Control Panel > Programs > Uninstall a Program.
  3. Uninstall |acquia-product:dam|CCConnector 1.0.
  4. If you want to retain existing InDesign plug-ins, do not use the uninstaller, but go to \C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\|acquia-product:dam|CCConnector and delete the folder.
