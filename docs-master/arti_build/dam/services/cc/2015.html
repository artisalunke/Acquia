After ensuring that you meet the installation requirements, to install the Adobe Creative Cloud (CC2015) connector, complete the following steps:

  1. Exit all Adobe programs.

  2. Use the appropriate procedure to install the Connector, based on your operating system:

    • MacOS

      1. Download and install the ZXPInstaller application. This application uses the same Adobe installation subsystem as Adobe AddOns.

        1. Go to zxpinstaller.com

        2. Click Download for OS X.

        3. Click the .dmg file in the download bar at the bottom of your browser.

        4. When the ZXPINstaller window displays, drag the ZXPInstaller to your Applications folder.

        5. Open ZXPInstaller from your Applications folder. (ZXPInstaller is not currently available on the Mac App Store, so your security settings may not allow you to open it. To change your settings, open your System Preferences and click Security & Privacy. Under âAllow apps downloaded from:â select Anywhere. (Some companies require IT to change this setting for you.)

        6. Download the .zxp file.

          Important

          Do not double-click the .zxp files or attempt to install using the Adobe CC Extension Manager â it will corrupt your installation. Adobe CC Extension Manager was discontinued with the CC 2014 release.

      2. Install the .zxp file, by either dragging the file into the window, or by clicking in the window and then selecting the .zxp file.
        ZXPInstaller will start installing the CC extensions in the .zxp file.
        If the installation fails, be sure that you have Adobe Creative Cloud, and not a version of Adobe Creative Suite. The Connector is not compatible with Creative Suite.

    • Microsoft Windows

      1. Download the WebDAMCCConnector.exe file.
      2. Click the .exe file in the download bar at the bottom of your browser. You may get a âWindows protected your PCâ warning because itâs an unrecognized file â if so, click More info then Run anyway.

  5. After you open the panel, enter your Acquia DAM credentials.
