Profile Manager¶

Profile Manager is a component of Acquia Lift.

The Profile Manager unifies person profile data collected from your websites, both Drupal and non-Drupal, and optionally can integrate data from other sources, such as email or a CRM.

Using Profile Manager and the information that you collect, you can create a high-resolution profile of your visitors, create and manage your segments and goals, run reports and analytics against your data, and administer how data is collected and mapped.

QUICK LINKS:   Managing segments  | Managing events  | Default segment criteria  | Release notes

Signing in to Profile Manager¶

To sign in to the Profile Manager interface and manage your campaigns and visitor information, visit the URL appropriate for your local Profile Manager Admin region:

  • Americas – us-east-1.lift.acquia.com
  • Europe, Middle East, and Africa – eu-central-1.lift.acquia.com
  • Asia-Pacific and Japan – ap-southeast-2.lift.acquia.com

If you are a Acquia Lift - Premium or Acquia Lift Starter - Premium subscriber, the Profiles API reference websites are available at the following URLs:

  • Americas – us-east-1-api.lift.acquia.com
  • Europe, Middle East, and Africa – eu-central-1-api.lift.acquia.com
  • Asia-Pacific and Japan – ap-southeast-2-api.lift.acquia.com

Information about your local Profile Manager Admin region is normally provided to you with your keys. Contact Acquia Support if you need assistance connecting to either website.

Password policy¶

When you first sign in to Profile Manager, you may be asked to change your password. Profile Manager uses the same password security policy as Acquia Cloud.

For a comprehensive explanation of the rules employed by Profile Manager regarding password complexity, see Realistic password strength estimation.

Adding tracking codes to your webpages¶

Drupal websites that use the Experience Builder can add tracking code to their pages by creating slots in Acquia Lift.

If your website uses a content management system (CMS) other than Drupal, you can add a simple JavaScript code snippet to your website, which will insert personalized content directly into your website, regardless of the content management system that you use. Users with Acquia Lift version 8.x-3.0 or greater can get started by following the steps in Using Acquia Lift with a non-Drupal website.

After you start collecting information about how your visitors interact with your website, you can review and segment those interactions by using Profile Manager.

Note for Drupal 7 users

If you use Drupal 7 as your website’s CMS, instead of manually adding the JavaScript tracker code to each of your webpages, we recommend that you use the included |acquia-product:alt| modules to send your website’s visitor interactions to the Acquia Lift service.
