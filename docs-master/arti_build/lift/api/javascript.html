Acquia Lift JavaScript API reference¶

Regardless of your website’s content management system (CMS), simply add the JavaScript tracking script to your website’s webpages. After you do so, you can then use the Acquia Lift JavaScript API’s functions to send additional information.

Note

We recommend that you install and use the Drupal modules to send information to Acquia Lift. If you use the Experience Builder modules, you do not need to use the JavaScript API.

The following JavaScript functions are available for your use:

Function Description
capture Capture a visitor’s page view
captureIdentity Capture specific visitor information
setDoNotTrack Enable or disable the tracking of visitors
updatePerson Update Person information without adding new records
