Configuring the Content Hub clientÂ¶

Installing Content Hub

  • Install
  • Configure

After you have installed the Content Hub client modules, configure the user permissions to administer and use the service, and then use the Drupal admin interface to configure them to connect your website to the Content Hub repository.

Configuring user permissionsÂ¶

To manage Content Hub settings (including the values on the Connections that that control your Hub assignment), signed-in users must have the appropriate permissions. To set the access permissions for Content Hub, complete the following steps:

  1. In the admin menu, go to People, and then click the Permissions tab.

  2. Assign the appropriate permissions to your websiteâs roles from the following list:

    • Content Hub: Administer content hub
    • Acquia Content Hub Connector: Administer content hub connector

    By default, all users with the Administrator role have these permissions.

  3. Click Save permissions to save your permission changes.

Configuring content sharing (publishing)Â¶

Next, see Sharing content to learn how to use the Entity Configuration and Settings tabs to configure how you share or publish content from your website to Content Hub, where it can be discovered and used by other members of your content network.

To ensure the data from your production environment is not corrupted by data from non-production environments, configure per-environment settings.

For a step-by-step video tutorial introducing you to personalization with Acquia Lift, including configuring content sharing, see Creating Personalized Experiences with Acquia Lift.

Disconnecting from Content HubÂ¶

Content Hub allows you to disconnect your website from Content Hub. It can be reconnected at a later time, if desired. Any content already published to the hub from the website you are disconnecting will remain on the hub, but it will be orphaned. Reconnecting your website to the hub will not reconnect the copy of the content on the hub to its original source.

Depending on your installed version of Drupal, use the one of the following methods to disconnect your website from Content Hub:

  • Drupal 7
    1. In the Admin menu, go to Configuration > Content Hub connector.
    2. If it is not already selected, click the Connection page.
    3. Click Disconnect.
  • Drupal 8 - Use one of the following Drush commands:
    • acquia-contenthub-disconnect-site
    • acquia-contenthub-webhooks
