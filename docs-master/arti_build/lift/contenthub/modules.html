Modules to use with Content Hub¶

The following contributed modules can be used with Content Hub, either with no modifications or with some additional modules or patches. Websites with complex or circular content models may experience unexpected issues and should contact Support if these issues occur.

Fully supported modules¶

The following modules work with Content Hub without any modifications:

  • ECK (Entity Construction Kit) versions 7.x-2.0-rc7 and greater
  • Paragraphs version 8.x - Supports both import and export

Modules requiring modifications¶

The following modules work with Content Hub, but require additional modules or patches to do so:

  • ECK (Entity Construction Kit) version 8.x - Requires that you grant anonymous users the View any [entity_type] entities permission

  • Field Collection versions earlier than 7.x-1.0-beta12 require the patch from this issue

  • File Entity (fieldable files) versions 7.x and 8.x - Files previously uploaded with this module cannot be syndicated until you run cron, which updates the bundle value for uploaded files.

  • Paragraphs version 7.x - Requires patch Because Content Hub supports only entities that have UUID, this module also requires the Paragraphs UUID module, which can be found in this sandbox.

  • Shield version 8.x - Requires both a patch from this issue on Drupal.org and adding the following configuration line to settings.php:

    $config['shield.settings']['paths'] = '/acquia-contenthub/*';

  • Workbench Moderation versions 7.x-1.4 and greater - Requires patch

Unsupported modules¶

The following modules are unsupported and may cause problems when used with Content Hub:

Drupal version Unsupported modules
Drupal 7
  • Brightcove Media version 7.x-1.5
Drupal 8
  • Drupal 8 Experimental modules
    None of these modules are recommended for use. For example, the Content Moderation experimental module causes errors when used with Content Hub — use Workbench Moderation instead.
  • Field Collection - Acquia recommends that you instead use the Paragraphs module.
  • File Entity
  • Metatag
