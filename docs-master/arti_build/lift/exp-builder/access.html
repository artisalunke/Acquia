Setting up access to the Experience Builder¶

Installing Experience Builder

  • Install
  • Access
  • Content

To personalize your website with Experience Builder, you and your users must be able to access the Experience Builder sidebar. Use the procedures on this page to set up and activate your Experience Builder bookmark.

Acquia Lift link¶

Drupal 8 users with the permissions:

  • Access Acquia Lift links
  • Access toolbar

are able to click the Acquia Lift link in the toolbar to activate the Profile Manager toolbar. This link will appear only on pages that support the Experience Builder.

Chrome extension¶

Google Chrome users can install a browser extension to enable the Acquia Lift toolbar.

  1. Download and install the extension.
  2. When visiting a page that uses Acquia Lift, click the rocket icon in your extension bar.

This activates the Profile Manager administrative login, and then the toolbar.

Setting up a bookmark¶

The Profile Manager toolbar is supported for Google Chrome and Mozilla Firefox, and can be used for Drupal or non-Drupal users. Select one of the following methods and follow its instructions to set up an Experience Builder bookmark for your use:

  • Google Chrome
    1. Right-click the bookmarks toolbar.
    2. Click Add Page.
    3. Add a Name like Lift Admin UI bookmarklet.
    4. Add the bookmark code in the URL.
    5. Click Save.
  • Firefox
    1. Right-click the bookmarks toolbar.
    2. Click New Bookmark.
    3. Add a Name like Lift Admin UI bookmark.
    4. Add the bookmark code in the Location.
    5. Click Add.

Bookmark code¶

Enter the following code in the appropriate place in the bookmark.

javascript:(function() { if (window.AcquiaLiftPublicApi) { window.AcquiaLiftPublicApi.activate(); } else { alert('Sorry, Acquia Lift could not be found'); } })();

Activating the Acquia Lift toolbar¶

After setting up and saving your bookmark, complete the following steps:

  1. Select a customizable page, such as your website’s homepage.
  2. Click the bookmark that you created. Acquia Lift displays a sign-in page for the Profile Manager administrative interface.
  3. Sign in using an Profile Manager account in the Administrators group.

After you sign in to Profile Manager, a sidebar slides out from the right side of the browser window. This sidebar is used for configuring Experience Builder operations, including creating slots and rules.

Click the three vertical dots to open or close the toolbar.
