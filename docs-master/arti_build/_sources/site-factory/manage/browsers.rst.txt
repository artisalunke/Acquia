Supported web browsers for |acquia-product:edg|
===============================================

|acquia-product:edg| supports the current and prior major release of the
following web browsers, both for website development and display:

-  `Chrome <https://www.google.com/chrome>`__
-  `Firefox <https://www.mozilla.org/firefox/>`__
-  `Internet
   Explorer <http://windows.microsoft.com/en-US/internet-explorer/download-ie>`__
-  `Safari <https://www.apple.com/safari>`__


Additional browser requirements
-------------------------------

|acquia-product:edg| requires that you enable the following features in
your supported browser:

-  JavaScript
-  Cookies

   .. note::  

      Safari users must configure their browsers to not block cookies from
      third parties or advertisers. To do this:

      #. In Safari, go to **Safari > Preferences**.
      #. Click the **Privacy** tab.
      #. In the **Cookies and website data** section, select the **Always
         allow** option.

         As an alternative, you could select **Allow from current website
         only** if you want to manage your cookies on a website-by-website
         basis.

      #. Close the Preferences window.
