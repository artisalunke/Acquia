Feature
~~~~~~~~~~~~~~

-  **Introducing |acquia-product:ch| API version 2** |br| 
   The |chapi|_ 
   has been updated to version 2, and includes an updated
   ``/v2/register`` method that requires a site ID that's included in
   the |acquia-product:cha| Profile Manager. The previous version 1
   method required only a custom site name without any restrictions.
   All commands from version 1 of the API remain supported.
