Creating and managing groups
============================

To easily assign `roles </journey/admin/users/roles>`__ to many
`users </journey/admin/users>`__ at once, |acquia-product:aj| allows you
to create a *group*. You must have either the Organization *Owner* or
Project *Admin* permissions to create or manage a group. For more
information about roles, see `About |acquia-product:aj|
roles </journey/admin/users/roles>`__.

Accessing groups
----------------

To view, create, or modify groups, complete the following steps:

#. Sign in to |acquia-product:aj|.
#. In the top menu, click **Admin**.
#. In the left menu, click **Groups**.

   |Journey Groups link|

|acquia-product:aj| will display any available groups. You can also use
this webpage to create a new group, or to add users or projects to any
of your currently existing groups.

Creating a group
----------------

To create a group, complete the following steps:

#. Navigate to your **Groups** page as described in `Accessing
   groups <#access>`__.
#. Click the plus icon ( **+** ) to open the **Add Group** dialog box.

   |Add a new group|

#. In the **Organization** list, click the organization that you want to
   use for your group.
#. Enter a descriptive name for your group in the **Name** field.
#. Click **Create Group** to save your group.

Your new group will appear in the groups list.

Adding a user to a group
------------------------

To associate a user with a group, complete the following steps:

#. Navigate to your **Groups** page as described in `Accessing
   groups <#access>`__.
#. Click the name of the group to which you want to assign a user.
#. Click **Add User**. |acquia-product:aj| will display the **Add
   Users** dialog box.
#. Select one or more users that you want to add to the group.
#. Click **Add Selected Users**.

The group will now display the users you selected as members of the
group.

Adding a group to a project
---------------------------

To associate a group with a project, complete the following steps:

#. Navigate to your **Groups** page as described in `Accessing
   groups <#access>`__.
#. Click the name of the group you want to associate with a project.
#. Click **Add Project**. |acquia-product:aj| will display the **Add
   Projects** dialog box.
#. Select a project.
#. Click **Add Selected Projects**.

   |Group administrative interface|
