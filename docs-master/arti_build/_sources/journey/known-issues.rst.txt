Known issues in |acquia-product:aj|
===================================

-  **Browser 'back' button is disabled**
   When navigating between pages in |acquia-product:aj|, using your
   browser's back button to return to a prior page does not work. Use
   the navigation provided within |acquia-product:aj| to navigate
   between pages.

If you encounter any difficulties with your |acquia-product:aj|
subscription, `contact Acquia Support </support#contact>`__ for
assistance.
