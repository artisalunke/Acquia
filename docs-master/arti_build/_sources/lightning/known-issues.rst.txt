Known issues in |acquia-product:ld|
===================================

This page provides additional information about specific known issues
with the |acquia-product:ld| Drupal distribution.

*At this time, there are no known issues that require additional
information.*

For general issues that are known about the |acquia-product:ld| Drupal
distribution, see the |acquia-product:ld| 
`issue queue <https://www.drupal.org/project/issues/lightning>`__ on
Drupal.org.

If you encounter any difficulties with the |acquia-product:ld| Drupal
distribution, `contact Acquia Support </support#contact>`__ for
assistance.
