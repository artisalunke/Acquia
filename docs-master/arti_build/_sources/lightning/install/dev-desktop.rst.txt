.. |devdesktop| replace:: \ |acquia-product:add|\ 
.. _devdesktop: https://www.acquia.com/products-services/dev-desktop

If you want to test or develop your websites using a desktop Mac or
Windows-based computer, you can install |devdesktop|_,
a completely functional local Drupal environment that contains all of
the required components for |acquia-product:ld|.

For information about downloading and installing |acquia-product:add|,
see `Installing Dev Desktop </dev-desktop/install>`__.

Installing a new |acquia-product:ld| site
-----------------------------------------

After you have installed |acquia-product:add|, for information about how
to add Drupal websites for your use, see `Getting started with
Dev Desktop </dev-desktop/start>`__. Be sure to select
**Lightning** from the list of available distributions.


|Select the Lightning distribution|

After the distribution has downloaded, 
`complete the installation </dev-desktop/start/new#complete>`__. For information about
features available during the installation process, see 
`Managing the additional Lightning features </lightning/install/features>`__.

Adding a local installation to |acquia-product:add|
---------------------------------------------------

If you have previously installed |acquia-product:ld| on your local
server, you can add it to |acquia-product:add| by 
`importing the local website </dev-desktop/start/local>`__.
