Installing Lightning
====================

.. toctree::
  :hidden:
  :glob:

  /lightning/install/*

If you already have a local or hosted environment available for your use
that meets `Drupal 8's system
requirements <https://www.drupal.org/requirements>`__, including working
versions of Apache, MySQL, and PHP, you're ready to install
|acquia-product:ld|.

Platform installation instructions
----------------------------------

For a list of installation instructions customized for your platform,
select the environment that used to host your |acquia-product:ld|
server:

- `Acquia Cloud </lightning/install/cloud>`__ |br| 
  Acquia's open cloud platform for Drupal
- `Acquia Dev Desktop </lightning/install/dev-desktop>`__ |br| 
  Create Drupal websites on your local computer
- `Local hosting </lightning/install/local>`__ |br| 
  Install the |acquia-product:ld| distribution on your local server

Developer tools
---------------

- `Acquia BLT <https://blt.readthedocs.io/en/latest/>`__ |br| 
  |acquia-product:blt| provides an automated layer for building, testing, and launching Drupal websites that are based on |acquia-product:ld|.
