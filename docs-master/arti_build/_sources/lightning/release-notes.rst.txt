Lightning 8.x-3.1.3
--------------------------

*25 April 2018*

This release of the |acquia-product:ld| Drupal 8.5-based distribution
(`download <https://www.drupal.org/project/lightning/releases/8.x-3.103>`__)
contains several changes and fixes, including the resolution of
`SA-CORE-2018-004 <https://www.drupal.org/sa-core-2018-004>`__.

For information about its specific changes, including Drupal.org issue
numbers and GitHub pull requests, see the `distribution release
notes <https://www.drupal.org/project/lightning/releases/8.x-3.103>`__
on Drupal.org.
