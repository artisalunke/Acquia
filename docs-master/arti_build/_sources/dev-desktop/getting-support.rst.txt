Getting support for |acquia-product:add|
========================================

If you have problems with |acquia-product:add|, be sure to:

-  Review the `Known Issues </dev-desktop/known-issues>`__ and
   `Troubleshooting </dev-desktop/troubleshooting>`__ pages for possible
   workarounds.
-  If you have a bug or feature request to report, see `Reporting
   issues, feature requests, and bugs </dev-desktop/reporting>`__.

If you have an |acquia-product:aqs| that entitles you to file support
tickets, you can `contact Acquia Support </support#contact>`__ about
|acquia-product:add| support questions.
