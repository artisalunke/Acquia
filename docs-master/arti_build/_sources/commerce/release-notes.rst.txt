Features
~~~~~~~~

- **Magento integration** |br|
  The initial release of Acquia Commerce Manager integrates with Magento's
  e-commerce platform to enable your website to create full user experiences.
  Integration includes features such as:

  - Categories and products synchronization
  - Cart management
  - Checkout flow
  - User management

  See the `Acquia Commerce Manager documentation </commerce>`__ or contact your
  account manager for additional details.
