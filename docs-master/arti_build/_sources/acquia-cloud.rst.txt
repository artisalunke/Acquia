|acquia-product:ac| is a Drupal-tuned application lifecycle management suite with a complete infrastructure to support Drupal deployment workflow processes from development and staging through to production. |acquia-product:ac| includes powerful developer user interfaces, secure server access using SSH, and automated deployment from a version-controlled code repository. It runs on proven open-source technologies that we've selected, tested, and optimized for Drupal. With |acquia-product:ac|, Acquia delivers comprehensive Drupal infrastructure support from a single vendor.

|acquia-product:ac| gives you:

-  Easy installation of any Drupal distribution
-  Great developer workflow tools that let you easily deploy content with drag-and-drop functionality across separate development, staging, and production environments
-  Burstable, elastic cloud resources that let you scale your servers on demand

|acquia-product:ac| supports businesses of any size with a variety of subscription levels:

-  *Acquia Cloud Professional* provides a self-service environment built for developing and deploying one or more production websites.
-  *Acquia Cloud Enterprise* delivers a fully managed, scalable, multiserver service, with high availability.

.. _access:

Accessing the |acquia-product:ui|
---------------------------------

After you create an |acquia-product:ac| subscription, visit the `Acquia Cloud interface <https://cloud.acquia.com/>`__ and sign in with your email address and Acquia password.

|Product interface|

Select an application and then click **Dev** to view details about your application's Development environment.
