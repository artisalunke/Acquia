Monitoring your applications and infrastructure
===============================================

.. toctree::
   :hidden:
   :glob:

   /acquia-cloud/monitor/*

|acquia-product:ac| provides a variety of tools you can use to monitor
your applications and the infrastructure they run on. These tools
include:

-  `Logs </acquia-cloud/monitor/logs>`__
-  `Stack Metrics </acquia-cloud/monitor/stackmetrics>`__
-  `Uptime monitoring </acquia-cloud/manage/uptime>`__
-  `Application monitoring tools </acquia-cloud/monitor/apm>`__
-  `Acquia Insight </acquia-cloud/insight>`__

Read the linked topics for information about how to use these tools to
monitor and troubleshoot issues.
