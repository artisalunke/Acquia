Choosing a Drupal distribution
==============================

A Drupal distribution is a curated package of Drupal software, including Drupal core, Drupal contributed modules, themes, libraries and installation profiles that together provide a Drupal solution. Although you can `import most Drupal distributions </acquia-cloud/create/import>`__ into |acquia-product:ac|. One-click selection is provided for the following popular Drupal distributions:

.. list-table::
 :widths: 25 75
 :header-rows: 1

 * - Distribution
   - Description
 * - |Lightning logo|
   - Lightning is an Acquia-curated distribution of the "best of Drupal." Designed to maximize development and accelerate website building, Lightning provides a lightweight framework and foundation for enterprise-grade Drupal 8. `Learn more </lightning>`__.
 * - |Drupal 8 core|
   - This is the latest Drupal 8 core package as posted on `Drupal.org <https://www.drupal.org>`__.

     .. note::

         When first installing this distribution, you will be asked for a database password; leave the password field empty, and then click **Save and Continue**.
 * - |Drupal 7 core|
   - This is the official Drupal 7 core package as posted on `Drupal.org <https://www.drupal.org>`__. Drupal 7 core comes bundled with a variety of modules and themes to give you a good starting point to help build your site. Drupal core includes basic community features, like blogging, forums, and contact forms, and you can easily extend it by downloading other contributed modules and themes.
 * - |Commerce Kickstart|
   - Commerce Kickstart, sponsored by `Commerce Guys <https://commerceguys.com/>`__, is the quickest way to get up and running with Drupal Commerce. It includes the latest versions of Drupal core, Drupal Commerce, and the other modules it depends on. It also gives you a set of options to create a fully-featured demo store out of the box, complete with an attractive theme, catalog and search configuration, and a custom back office interface. `Learn more. <https://www.drupal.org/project/commerce_kickstart>`__
 * - |DKAN|
   - DKAN is an open data platform that allows governments, nonprofits and universities to easily publish data to the public. `Learn more. <https://www.drupal.org/project/dkan>`__
 * - |govCMS|
   - The govCMS distribution is built using Drupal 7 core, along with additional software such as themes, modules, libraries, and installation profiles specific to Australian government websites. `Learn more. <https://www.govcms.gov.au/>`__
 * - |Open Atrium|
   - Open Atrium is an open source platform designed specifically to make great teams communicate better. An intranet in a box with a blog, a wiki, a calendar, a to-do list, a shoutbox, and a dashboard to manage it all. Itâ€™s also completely customizable. `Learn more. <https://www.drupal.org/project/openatrium>`__
 * - |Web Experience Toolkit|
   - The Drupal WxT (Web Experience Toolkit) distribution has been tailored for organizations that need to comply with standards for accessibility and bilingualism, and integrates extensively with the WET-BOEW jQuery Framework. `Learn more. <https://www.drupal.org/project/wetkit>`__
