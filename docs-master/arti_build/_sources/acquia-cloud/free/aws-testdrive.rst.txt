Getting started with an AWS Test Drive
======================================

.. toctree::
   :hidden:
   :glob:

   /acquia-cloud/free/aws-testdrive/*

The `AWS Drupal Test Drive
page <https://aws.amazon.com/testdrive/drupal/>`__ offers a collection
of Drupal distributions that provide customers with the ability to
easily explore the benefits of using Drupal. The following pages
describe how to get started with Drupal distributions on
|acquia-product:acfs|:

-  `Installing OpenPublic from AWS Test
   Drive </acquia-cloud/free/aws-testdrive/openpublic>`__
-  `Installing Acquia Demo Framework from AWS Test
   Drive </acquia-cloud/free/aws-testdrive/demo-framework>`__
-  `Installing DKAN from AWS Test
   Drive </acquia-cloud/free/aws-testdrive/dkan>`__
