Module listing for Lightning¶

To help you develop and publish your website, the Lightning distribution uses a collection of included modules. These modules, provided by Acquia and the Drupal community (and by Drupal itself), provide a base level of integrated features that you can use as you develop your website.

Note

If the Lightning distribution doesn’t contain a module that you require, simply find and download the module from Drupal.org and then install it on your website. For information about how to install a module, see Installing modules and themes on Drupal.org.

Drupal core modules¶

Regardless of the version of Drupal that it’s based on, Lightning installs a group of Drupal core modules, most of which can be uninstalled if desired. These modules provide the base level of functionality for your website, including the areas of content creation, display, and website administration.

Module versions¶

To display detailed information about the included modules versions and any installed patches, visit the Lightning distribution page on Drupal.org and then click the release note link for the current version in the Downloads section.
