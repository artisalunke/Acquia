Supported web browsers for Acquia Cloud Site Factory¶

Acquia Cloud Site Factory supports the current and prior major release of the following web browsers, both for website development and display:

  • Chrome
  • Firefox
  • Internet Explorer
  • Safari

Additional browser requirements¶

Acquia Cloud Site Factory requires that you enable the following features in your supported browser:

  • JavaScript

  • Cookies

    Note

    Safari users must configure their browsers to not block cookies from third parties or advertisers. To do this:

    1. In Safari, go to Safari > Preferences.

    2. Click the Privacy tab.

    3. In the Cookies and website data section, select the Always allow option.

      As an alternative, you could select Allow from current website only if you want to manage your cookies on a website-by-website basis.

    4. Close the Preferences window.
