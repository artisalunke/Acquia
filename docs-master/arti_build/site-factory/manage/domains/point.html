Pointing your domain name¶

Custom domains

  • Prepare
  • Point
  • Verify
  • Email
  • Complete

After you have your new domain name (based on our domain name registrar recommendations), point your domain name, subdomain name, or path-based domain name to your website running on Acquia Cloud Site Factory.

Note

If you are adding an additional custom domain name to an existing Acquia Cloud Site Factory subscription with a custom domain name, before you point the additional domain name to your subscription you must first add a 301 redirect for that domain to the .htaccess file for your website’s codebase. Failing to implement the redirect may cause HTTP 404 Not Found error messages to be cached for your new custom domain.

To point your domain name to a website, use one of the following procedures, depending on whether you are adding a domain to a website or a site collection, and the method supported by your domain name registrar:

  • Adding a domain to a website
    • Using CNAME with your website (preferred)
    • Using the A Record with your website
  • Adding a domain to a site collection
    • Using CNAME with your site collection (preferred)
    • Using the A Record with your site collection

Be patient, as this step can take up to 24 hours for DNS entries to propagate across the Internet.
