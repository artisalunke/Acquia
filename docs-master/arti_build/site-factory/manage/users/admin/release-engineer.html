Release engineer role¶

The release engineer role on Acquia Cloud Site Factory is assigned to privileged PaaS subscription users who have the responsibility for updating the production environment with new and updated code.

Note

The account with the release engineer role must also have the platform admin role to use the Administration admin menu link to access its available actions.

To request that a user account be associated with this role, contact Acquia Support.

Users that are assigned the release engineer role can perform the following actions on Acquia Cloud Site Factory:

  • Sign in to Acquia Cloud Site Factory
  • View the site update status for an environment
  • View the environment’s staging deployment status
  • Release code updates to the production environment
