To help you manage your Acquia Cloud Site Factory websites and their user accounts, Acquia Cloud Site Factory includes the Site Factory Management Console, a browser-based website management interface that allows you to effectively organize and work with many websites and users.

Using the Site Factory Management Console, you can administer user accounts, perform website management tasks, and administer overall Acquia Cloud Site Factory settings.

For instructions for signing in to the Site Factory Management Console, see Signing in to Acquia Cloud Site Factory.

Visit Acquia Academy (sign-in required) for the Setting Up Acquia Cloud Site Factory video tutorial. Sign-in required.

User administrationÂ¶

User management on Acquia Cloud Site Factory involves both the roles, access, and permissions for your Site Factory Management Console, and the roles, access, and permissions for the individual Acquia Cloud Site Factory-hosted websites. The following user management tasks are available for your use:

  • Manage users with access to your Site Factory Management Console
  • Assign roles to users when they sign in
  • Configure two-factor authentication in the Site Factory Management Console
  • Select an owner for a specific website
  • Manage policies for password strength

For information about the permissions matrix for roles on your Site Factory Management Console, see Site Factory Management Console roles.

Website management tasksÂ¶

Creating, maintaining, and configuring the individual websites hosted by Acquia Cloud Site Factory is a straightforward process in the Site Factory Management Console, allowing you to complete the following common tasks:

  • Stack configuration and management
  • Create new websites or groups of websites
  • Duplicate existing websites
  • Backup management
  • Theme repository management
  • Domain management

For a full list of tasks, see Site actions and organization.

Platform administration tasksÂ¶

As part of maintaining your platform, you need to evaluate how Acquia Cloud Site Factory is handling your websites, both for normal use and during code deployments and updates. Also, periodically you may need to make configuration changes that can affect all of your Acquia Cloud Site Factory-hosted websites.
