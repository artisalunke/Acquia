Release notes - Acquia Commerce Manager¶

Looking for the latest and greatest new features and changes to Acquia Commerce Manager? Read on and check back regularly to see what else we’ve done.

Acquia Commerce Manager¶

27 Aug 2017

Features¶

  • Magento integration
    The initial release of Acquia Commerce Manager integrates with Magento’s e-commerce platform to enable your website to create full user experiences. Integration includes features such as:

    • Categories and products synchronization
    • Cart management
    • Checkout flow
    • User management

    See the Acquia Commerce Manager documentation or contact your account manager for additional details.
