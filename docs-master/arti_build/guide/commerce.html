Acquia Commerce Manager Product Guide¶

For additional information about Acquia Commerce Manager, see its product documentation.

Last updated: December 11, 2017

Acquia Commerce Manager connects the Acquia platform to the Customer’s eCommerce platform so that merchandisers can quickly and securely create unique tailored online shopping experiences that best represent their brand.

An Acquia Commerce Manager Subscription consists of the following components:

  1. Either a single website on Acquia Cloud Enterprise or number of websites permitted within the relevant Acquia Cloud Site Factory subscription, in a PCI Virtual Private Cloud.
    For specific entitlements related to the portion of Acquia Commerce Manager (based on the corresponding level of Acquia Commerce Manager), see the relevant sections of the Acquia Product and Services Guide for Acquia Cloud Enterprise or Acquia Cloud Site Factory.
  2. Commerce Connector Service for use with Lightning for Commerce
    This is a SaaS (Software as a Service) that connects eCommerce stores to Drupal applications running on either Acquia Cloud Enterprise or Acquia Cloud Site Factory, and is subject to the following:
    • 2.1 Commerce Connector Service
      The Commerce Connector Service is to be used the latest version of the open source Lightning for Commerce sub-distribution of Drupal and the latest version of eCommerce platform extension as available for specific platforms in their relevant marketplaces.
    • 2.2 5 Terabytes of data
      Up to 5 Terabytes of data can be transferred through the Commerce Connector Service per year. Customers trending towards maximum data usage must work with Acquia Support to address application issues causing the unnecessary transfer of large volumes of data.
    • 2.3 Data Centers, Backup, and Disaster Recovery for |acquia-product:ccs|:
      The infrastructure supporting the Commerce Connector Service is physically remote from Acquia’s office facilities. The Commerce Connector Service environment consists of major Regions and Availability Zones. Availability Zones are separate yet interconnected data centers within major Regions in the global infrastructure. Commerce Connector Service utilizes a highly available redundant architecture that distributes replicated redundant server types (load balancing, caching, web and database servers) across multiple Availability Zones within the same Region. Acquia will use commercially reasonable efforts to restore the services in an alternate Availability Zone within the same Region in the event service in Customer’s assigned Availability Zone is severely impacted. The Commerce Connector Service transmits data and does not process or store it as such there is no data to backup performed as part of the service.
    • 2.4 Service Level Policy
      Commerce Connector Service inherits the SLA of the applicable Acquia Cloud Enterprise or Acquia Cloud Site Factory subscription purchased with the Acquia Commerce Manager.
  3. Payment Card Industry Data Security Standard (PCI DSS) Compliant Environment (“PCI Environment”)
    Acquia provides a PCI-compliant hosting environment as part of Acquia Commerce Manager. PCI DSS compliance applies to merchants and service providers that process, store, or transmit credit card data. PCI DSS is a multifaceted security standard that includes requirements for security management, policies and procedures, network architecture, software design, and other critical protective measures. This comprehensive standard helps organizations proactively protect credit card data that is transmitted or stored with this environment. Only those Customer applications built on the PCI Environment or via a dedicated PCI VPC Shield offering included with the applicable Acquia Commerce Manager subscription enjoy such PCI protections.
    Acquia’s PCI Environment is reviewed and validated by a PCI Security Standards Council authorized independent Quality Security Assessor (QSA) on an annual basis to against the then-current PCI-DSS standards applicable to a Level 1 service provider. Customers are responsible for assessing and implementing applicable PCI Compliance requirements for their applications they built on the PCI Environment.
  4. Support Services
    Customer’s subscription to Acquia Commerce Manager includes unlimited diagnosis support for Commerce Connector Service, as described in the Supportability Matrix in the Support Users’ Guide. The Customer can contact Acquia Support in accordance with both the Support Users Guide and the stated Urgency Levels. Initial response times for support requests vary on both the urgency level and Customer’s Subscription tier, as described in the Support Users Guide.
Acquia Inc. reserves the right to change the Products and Services Guide based on prevailing market practices and the evolution of our products. Changes will not result in a degradation in the level of services provided during the period for which fees for such services have been paid.
